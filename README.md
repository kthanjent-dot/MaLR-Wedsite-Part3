# MaLR-Wedsite-Part3
A modern, responsive landing page for **MaRL** cleaning and care products. The site includes a hero slider, product gallery, navigation, and interactive forms.

## Features
- Responsive navigation for desktop and mobile
- Interactive hero slider with controls
- Product cards and modern homepage sections
- Accessible form labels and live status messages
- Styled buttons and cards for a polished look

## Project Structure
- `Home.html` — homepage
- `products.html` — product information page
- `resellers.html` — reseller application page
- `partners.html` — partner inquiry page
- `Contact.html` — contact page
- `styles.css` — all styling
- `script.js` — interactivity and slider logic

## How to Use
1. Open any page directly in a browser, or use a local web server.
2. Browse the sections using the navigation menu.
3. Use the forms to submit inquiries.

## Accessibility Notes
- Buttons and links include clear focus styling.
- Form messages use live regions for screen reader feedback.
- Navigation includes ARIA labels for the menu toggle.
